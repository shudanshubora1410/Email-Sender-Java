# Email Sender Application (Core Java)

## Overview
This is a Core Java based Email Sender application developed as part of Java learning. The application allows users to send emails by providing recipient details, subject, and message body.

## Technologies Used
- Java
- JavaMail API
- Apache NetBeans IDE

## Features
- Send emails using SMTP protocol
- Simple and user-friendly interface
- Supports subject and message body
- Exception handling for email errors

## How to Run
1. Open the project in Apache NetBeans IDE
2. Configure email credentials in the source code
3. Build and run the application

## Learning Outcome
- Understanding of Core Java concepts
- Hands-on experience with JavaMail API
- Practical use of exception handling and external libraries

# Email Sender Application (Core Java)

## Overview
This is a Core Java based desktop application that allows users to send emails directly from the application using Gmail SMTP. The project demonstrates how Java can be used to automate email sending with support for multiple recipients and file attachments.

## Technologies Used
- Java
- Java Swing (GUI)
- JavaMail API
- SMTP (smtp.gmail.com)
- Apache NetBeans IDE

## Features
- Send emails using Gmail SMTP server
- Send emails to multiple recipients
- Add subject and message body
- Send emails with file attachments
- Simple and user-friendly graphical interface
- Password show/hide feature
- Exception handling for email errors

## How to Run
1. Open the project in Apache NetBeans IDE
2. Configure your email credentials in the source code
3. Build and run the application
4. Enter recipient email, subject, and message
5. Click **Send** to send the email

## Learning Outcome
- Understanding of Core Java concepts
- Hands-on experience with JavaMail API
- Working with SMTP protocol
- Practical use of Java Swing for GUI development
- Exception handling and file attachments in Java
